# 🎓 GTUOR – Virtual 3D AI Tutor System

## Overview
GTUOR (AI Tutor) is a 3D AI-powered teaching system designed for **Indian students and software professionals**.  
The virtual tutor behaves like a real human teacher — walking, talking, writing on a chalkboard, explaining, and adapting lessons dynamically based on student understanding.

## Features
- 🧠 Personalized adaptive learning flow  
- 🗣️ Lip-sync speech and gestures  
- ✍️ Chalk-style writing animation on blackboard  
- 🎮 Unity-compatible control logic  
- 📊 Student progress tracking  
- 🎓 Suitable for school & professional education  

## Tech Stack
- **Unity 3D** – 3D environment and animation  
- **C#** – Event-driven control logic  
- **GPT / Claude / Gemini API** – AI tutor intelligence  
- **TTS / STT** – Voice output and speech recognition  
- **Cursor IDE** – AI-assisted coding and repo management
